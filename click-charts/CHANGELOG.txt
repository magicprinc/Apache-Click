Changelog
=========

Version 1.2.0 - 2010-05-22
--------------------------

* Migrated controls to import resources through Elements and the
  getHeadElements method

* Aligned Charts with the Servlet 3.0 spec to deploy resources from the directory
  'META-INF/resources' instead of 'META-INF/web'. This release is compatible
  with Click 2.2.0 and up.

Version 1.0 - 2009-02-15
------------------------

* Initial release, Charts ported to Apache Click
