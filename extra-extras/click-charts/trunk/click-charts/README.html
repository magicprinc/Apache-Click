Click Charts Project

Click Charts is a port of the Chart controls found in pre Apache Click releases. The Chart controls had to be removed because LGPL libraries cannot be bundled with Apache Software Foundation distribution.

Please note Click versions 1.5.x and below shipped with the Chart controls so there is no need to use this library is pre Apache releases.

Only consider this library when using an Apache Click release.

Project structure

The project consists of the Charts library and an example webapplication.

Following modules are available:

  • [charts] - the charts library
  • [chart-examples] - example usage for the charts module

Other directories:

  • [dist] - build distribution directory
  • [documentation] - all documentation (javadocs too)
  • [lib] - libraries downloaded automatically with get-deps task.

Runtime Dependencies

Click Charts has the following runtime dependencies

  • click-2.2.0.jar or greater
  • click-extras-2.2.0.jar or greater

Compile time Dependencies

When compiling the project, the following JARs need to be present in the [lib] directory: (see Step 1. in the Building section below on how to automatically download the libraries)

  • click-2.2.0.jar or greater
  • click-extras-2.2.0.jar or greater
  • servlet-api-2.3.jar

Building

To build Click Charts, execute the following steps:

  1. get all dependencies (and download manually those jars not found in Maven repo), by running:

    ant get-deps

    If you are behind a firewall set your proxy configuration in build.properties and instead run:

    ant get-deps-proxy

  2. build all modules, controls sets, examples, by running:

    ant build-all

  3. (optional) build source packages

    ant build-sources

  4. (optional) build the entire distribution

    ant build-distribution
