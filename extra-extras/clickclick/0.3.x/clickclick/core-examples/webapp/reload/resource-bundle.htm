Resource Bundle Reload Demo

This page demonstrated that resource bundle properties can be reloaded on the fly. (You will need to have ClickClick Examples open in your IDE)

To see it in action, change the value of 'myproperty' in the resource bundle
'net/sf/clickclick/examples/page/reload/ResourceBundlePage.properties'. Make sure your IDE is setup to copy the updated 'ResourceBundlePage.properties' file to your web application build folder. Refresh the browser to see the new value.
