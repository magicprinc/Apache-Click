Changelog
=========

Version 1.3.0 - 2011-03-28
--------------------------

 * Fixed bug where the default date format pattern could not be overridden
   from click-page.properties. With this fix it is easy to define a new
   default date format pattern.

Version 1.2.0 - 2010-05-22
--------------------------

* Added support for min/max year range

* Migrated controls to import resources through Elements and the
  getHeadElements method

* Aligned Calendar with the Servlet 3.0 spec to deploy resources from the directory
  'META-INF/resources' instead of 'META-INF/web'. This release is compatible
  with Click 2.1.0 and up.

Version 1.0.2 - 2009-07-18
--------------------------

* Fixed bug that disabled keyboard input when calendar popup was destroyed.

Version 1.0.1 - 2009-06-15
--------------------------

* Changed the calendar setup to destroy the calendar instance when the popup is closed.
  This fixes a memory leak where calendar instances are not cleaned up [CLK-499].

Version 1.0 - 2009-02-15
------------------------

* Initial release, calendar ported to Apache Click
