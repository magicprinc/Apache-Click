Ajax4Click Project

The aim of this project is to provide Ajax utilities for Apache Click.

Project structure

The following modules are available:

  • [ajax4click] - common Ajax utilities and controls
  • [ajax4click-examples] - Ajax examples

Other directories:

  • [dist] - build distribution directory
  • [documentation] - all documentation (javadocs too)
  • [lib] - libraries downloaded automatically with get-deps task.

Dependencies

The following JARs need to be present in the [lib] directory:

  • click-core-2.3.0.jar
  • click-extras-2.3.0.jar
  • ajax4click-0.1.jar
  • servlet-api-2.3.jar

Building

To build Ajax4Click, execute the following steps:

  1. get all dependencies (and download manually to the /lib directory those jars not found in Maven repo), by running:

    ant get-deps

    build the project by running:

    ant build-all

  2. (optional) build source packages

    ant build-sources

  3. (optional) build the entire distribution

    ant build-dist
